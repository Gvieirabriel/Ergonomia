Operações Binárias

Neste pequeno curso você aprenderá os conceitos básicos de responsividade e como funciona o sistema de grids responsivos do framework Bootstrap.

  • Objetivo Principal: Ensinar operações básicas na base binária
  • Público Alvo: Universitários estudantes de tecnologia
  • Pré-Requisitos: Conhecimento básico em matemática
  • Duração: Aproximadamente 15 minutos

Introdução

Os computadores trabalham com um sistema matemático que utiliza dois caracteres para manipular qualquer informação, isso quer dizer que todas as operações o computador faz utilizando apenas dois caracteres, que por convenção são os dígitos “0” (zero) e “1” (um), daí podemos concluir que para 0 temos desligado, sem sinal, e para 1 temos ligado ou com sinal, esses são os dígitos binários ou somente bit (conjunção de duas palavras da língua inglesa binary digit), que é a menor unidade de informação dos computadores.
